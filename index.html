Installation:

  1. install BetterDiscord or Bandaged BetterDiscord
  2. download file ending with .theme.css from this site
  3. open discord navigate to settings > themes and press "Open Theme Folder" on top
  4. move .theme.css file to this folder
  5. in discord press ctrl+r, discord will restart
  6. enable theme in settings
  7. done!

Theme Description:

hover on image to see difference!

Makes servers on serverlist smaller allowing to fit more of them in same space
