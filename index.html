Themes Theme Addons

Installation:

  1. 1. install BetterDiscord
  2. 2. download file ending with .theme.css from this site
  3. 3. open discord navigate to settings > themes and press "Open Theme Folder" on top
  4. 4. move .theme.css file to this folder
  5. 5. in discord press ctrl+r, discord will restart
  6. 6. enable theme in settings
  7. 7. done!
