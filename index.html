Installation:

  1. install BetterDiscord or Bandaged BetterDiscord
  2. download file ending with .theme.css from this site
  3. open discord navigate to settings > themes and press "Open Theme Folder" on top
  4. move .theme.css file to this folder
  5. in discord press ctrl+r, discord will restart
  6. enable theme in settings
  7. done!

Theme Description:

hover on image to see difference!

Google Material Icons
replaces most if not all icons with Google's material icons
Bigger Youtube embed
increases size of youtube embed
Wider Discord settings
Widens Discord settings
Fit Serverlist
Makes servers on serverlist smaller allowing to fit more of them in same space
Multicolumn serverlist
for people with too many servers
you can adjust amount of columns with:
:root { --serverlist-column-count: 2; }
Transparent theme
Theme that's transparent and has multiple customisation options
:root { --background-color: #101010; --background-image: url("[image url here]"); --background-image-opacity: 0.3; --background-image-blur: 0px; --color-text: rgba(255, 255, 255, 0.5); --color-text-hover: rgba(255, 255, 255, 0.75); --color-text-muted: rgba(125, 125, 125, 0.4); --color-text-big-divider: rgba(125, 125, 125, 0.5); --color-main: rgba(255, 255, 255, 0.1); --color-main-hover: rgba(255, 255, 255, 0.2); --color-divider: rgba(0, 0, 0, 0); --color-big-divider: rgba(0, 0, 0, 0); }
